# 2D 木筏漂流生存游戏 V0.5 设计文档

版本目标：先做出一个最简单、能玩的半成品版本。  
玩法重点是：捞资源、扩建木筏、建造基础设施、烤鱼恢复饥饿。  
本版本没有口渴、生命值、鲨鱼、受伤、死亡、岛屿、天气、昼夜和粒子特效。唯一负面效果是饥饿。

---

## 1. 游戏定位

### 核心类型

2D 俯视角木筏漂流生存建造游戏。

### 核心体验

玩家站在小木筏上，用钩子捞取海上漂来的资源，通过资源合成和建造，让木筏慢慢变大，并用烧烤架烤鱼来维持饥饿值。

### V0.5 核心循环

```text
站在木筏上
→ 用钩子捞资源
→ 获得木板、塑料、树叶、漂流箱
→ 合成绳子、地板、烧烤架、储物箱、收集网
→ 扩建木筏
→ 从漂流箱获得生鱼
→ 用烧烤架烤鱼
→ 吃烤鱼恢复饥饿
→ 继续扩建和收集
```

## 5. 玩家规则

### 玩家移动

玩家只能在木筏地板上移动，不能下海。

### 玩家状态

本版本只有一个状态：饥饿值。

| 状态 | 数值范围 | 说明 |
|---|---:|---|
| 饥饿值 | 0 - 100 | 会随时间缓慢下降 |

### 饥饿规则

| 规则 | 数值 |
|---|---:|
| 初始饥饿值 | 100 |
| 每秒下降 | 0.3 |
| 吃烤鱼恢复 | +35 |
| 饥饿值最低 | 0 |

饥饿为 0 时，不扣血、不死亡，只产生轻微惩罚：

```text
玩家移动速度降低 30%
屏幕提示：太饿了，需要吃点东西
```

## 8. 建筑设计

### 木筏地板

| 建筑ID | 名称 | 用途 |
|---|---|---|
| raft_floor | 木筏地板 | 玩家站立、摆放建筑、扩建木筏 |
| raft_edge | 木筏边缘 | 视觉装饰，可选 |

### 功能建筑

| 建筑ID | 名称 | 用途 |
|---|---|---|
| build_grill_empty | 空烧烤架 | 未放入食物 |
| build_grill_cooking | 烧烤架烹饪中 | 生鱼正在烤 |
| build_grill_done | 烧烤架完成 | 可以取出烤鱼 |
| build_storage_closed | 储物箱关闭 | 默认状态 |
| build_storage_open | 储物箱打开 | 打开存储界面 |
| build_collection_net | 自动收集网 | 自动收集经过的漂浮资源 |

### 烧烤架规则

| 行为 | 规则 |
|---|---|
| 放入生鱼 | 消耗 item_raw_fish ×1 |
| 烹饪时间 | 8 秒 |
| 完成产物 | item_cooked_fish ×1 |
| 同时烹饪数量 | 1 条鱼 |

### 储物箱规则

| 项目 | 数值 |
|---|---:|
| 格子数 | 12 |
| 可存放物品 | 所有背包物品 |

### 收集网规则

收集网只能放在木筏边缘。  
当漂浮资源碰到收集网时，会自动进入玩家背包，或者进入临时缓存。

建议 V0.5 中直接进入玩家背包，减少系统复杂度。

## 9. 漂浮物生成规则

### 基础生成

海面从屏幕上方或右侧不断生成漂浮物，漂浮物沿固定方向移动。

| 漂浮物 | 权重 |
|---|---:|
| 木板 | 40 |
| 塑料 | 30 |
| 树叶 | 25 |
| 漂流箱 | 5 |

### 生成频率

| 项目 | 建议数值 |
|---|---:|
| 每次生成间隔 | 1.2 - 2.0 秒 |
| 漂浮物移动速度 | 60 - 90 px/s |
| 钩子最大距离 | 280 px |
| 钩子飞行速度 | 500 px/s |
| 钩子拉回速度 | 450 px/s |

---

## 10. 木筏扩建规则

初始木筏大小建议为 3×3。

### 建造规则

木筏地板只能建在已有木筏地板的上下左右相邻位置。  
建筑只能放在已有木筏地板上。  
收集网只能放在木筏外边缘。

### V0.5 目标

```text
把木筏扩建到 5×5
建造 1 个烧烤架
建造 1 个储物箱
建造 2 个收集网
```

达成后可以显示：

```text
木筏已经稳定下来啦！
V0.5 目标完成
```

## 14. 推荐第一版开发顺序

1. 画海面背景循环。
2. 放置 3×3 初始木筏。
3. 做玩家移动，只允许在木筏上移动。
4. 做漂浮物生成和移动。
5. 做钩子投掷、命中、拉回。
6. 做背包和资源数量。
7. 做合成绳子和木筏地板。
8. 做木筏扩建。
9. 做烧烤架、生鱼、烤鱼。
10. 做饥饿值下降和吃鱼恢复。
11. 做储物箱。
12. 做收集网。
13. 做 V0.5 完成目标检测。

## 15. V0.5 完成标准

当玩家完成以下目标时，V0.5 即可算作可玩版本：

- 玩家可以移动。
- 玩家无法走出木筏。
- 海上会生成漂浮资源。
- 玩家可以用钩子捞资源。
- 资源可以进入背包。
- 玩家可以合成绳子。
- 玩家可以扩建木筏。
- 玩家可以建造烧烤架。
- 漂流箱可以开出生鱼。
- 生鱼可以被烤成熟鱼。
- 烤鱼可以恢复饥饿。
- 玩家可以建造储物箱。
- 玩家可以建造收集网。
- 游戏能检测 5×5 木筏、1 个烧烤架、1 个储物箱、2 个收集网的目标。

## 16. 最终一句话版本

V0.5 是一个没有复杂惩罚的 2D 木筏漂流小游戏：玩家用钩子捞资源，扩建木筏，建造烧烤架和收集网，通过吃烤鱼维持饥饿，并最终把木筏建设成一个稳定的小海上基地。
